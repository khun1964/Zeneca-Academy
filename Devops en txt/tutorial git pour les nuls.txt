au début créer a new repositery

Avec New ou à droite dans le menu déroulant ou situé avec le + ( new repository).

Owner / Repository name.

example:

khun1964 / jboss

Aller dans le dossier à la racine du pc. Enclencher la console/ligne de commande & effacer puis écrire CMD.

git init

git commit -m C:\Users\khun1\Desktop\boule-de-haine-conciliante\jboss

à la requête : git remote add origin https://github.com/khun1964/jboss.git / la réponse remote origin already exists

git branch -M main

git push -u origin main
